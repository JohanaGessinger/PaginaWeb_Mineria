🌎 Cataratas del Iguazú – Distribución de Turistas por País

Este repositorio presenta una visualización sencilla sobre la cantidad de turistas que visitan las Cataratas del Iguazú según su país de origen.
Incluye una imagen representativa, una tabla con datos estadísticos y un gráfico en canvas que muestra la distribución de visitantes de países como Argentina,
Brasil, Estados Unidos, Alemania, Francia y otros. El proyecto utiliza HTML, CSS y JavaScript para organizar y representar la información de forma clara y atractiva.
